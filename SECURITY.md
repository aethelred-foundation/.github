# Security Policy

## Reporting a Vulnerability

Please do not use public issues or pull requests for security reports.

Send reports to **security@aethelred.io**.

Include:

- affected repository or organization surface
- impacted branch, tag, workflow, or profile asset
- reproduction steps or proof of concept
- impact summary
- any relevant logs or screenshots

If the report applies to a specific product repository, also include the repository link and affected release reference.
