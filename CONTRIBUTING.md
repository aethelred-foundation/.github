# Contributing

This repository is intentionally narrow in scope. Contributions here should be limited to organization profile assets, repository governance defaults, and validation workflows for this repository.

## Before opening a pull request

- confirm the change belongs in the organization configuration repo
- keep profile copy factual and aligned with the public protocol posture
- avoid adding repository-specific product logic here
- run the profile validation workflow or equivalent local checks when touching `profile/`

## Security

Security issues should be reported through `security@aethelred.io`, not through public issues or pull requests.
